Intelligent UI Helper / 智能 UI 导师

这是该仓库的 GitHub Pages 主页。内容依据 README 信息整理，包含 Figma 插件与 Web 导师双轨能力， 并提供演示 Deck 入口。

打开演示 Deck 查看 GitHub 仓库

Track A：Figma 插件

在 Figma 中进行导师式对话、文生图、图生图/编辑。核心目录：figma-ui-tutor/。

Track B：Web 画布导师

Vite + React + Konva 可视化编辑，后端结构化输出。核心目录：src/、server/、shared/。

模型与接口

Figma 插件侧默认为 SiliconFlow + Qwen；Web 侧可通过后端代理对接 Grok 结构化生成。

快速开始

安装依赖：npm install
开发启动：npm run dev
构建生产：npm run build
请先复制 .env.example 为 .env 并填写密钥。
